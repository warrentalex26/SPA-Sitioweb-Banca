# Sitio web SPA creado para mostrar aplicaciones del sector bancario, lo unico que falta es el responsive design y los :hover para los botones y los tarjetas, cabe destacar que los :hover son opcionales dependiendo de la creatividad de cada uno de nosotros

Esta pagina web esta hecha SIN LIBRERIAS!. Lo unico que ocupe fue google fonts para las fuentes, flexbox para el posicionamiento de los elementos, javascript ES6 y buenas prácticas con CSS. Además debes tener buen conocimiento de los position en CSS para entender los estilos
